# twinkle

for [jsstg2015](http://jp.wgld.org/jsstg/2015f/)

## install

```
git clone https://github.com/daishihmr/jsstg2015-twinkle.git
git submodule update
npm install
```

## build

```
grunt
```

## run

### on web browser

```
grunt web
```

### as application

run build/jsstg2015-twinkle/{PLATFORM}/jsstg2015-twinkle
